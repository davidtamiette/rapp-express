# Ápia Ambiental - Elaboração e Envio de RAPP Ibama

Página direcionada à captação de clientes para o serviço de elaboração e envio do RAPP (Relatório Anual de Atividades Potencialmente Poluidoras) do Ibama.

## Tecnologias

Este projeto utiliza:

- [Vite](https://vitejs.dev/)
- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)

## Como rodar o projeto localmente

Siga as instruções abaixo para executar o projeto em seu ambiente:

```sh
# 1. Instale as dependências.
npm install

# 2. Inicie o servidor de desenvolvimento.
npm run dev
```
